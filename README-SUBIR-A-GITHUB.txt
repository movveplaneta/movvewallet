MOVVE PLANETA - REDISEÑO CLARO

Archivos principales:
- index.html: página principal nueva
- styles.css: diseño claro sin verde neón y sin fondos negros
- script.js: menú responsive y animaciones suaves
- movve.html, presentacion.html, tutoriales.html, inversion.html, crecimiento.html, comunidad.html: páginas enlazadas
- myfxbook.html, onequity.html, wikifx.html: páginas de recursos
- sitemap.xml y robots.txt: SEO básico

Cómo subir a GitHub Pages:
1. Abre tu repositorio.
2. Sube todos estos archivos a la raíz del proyecto.
3. Si ya tienes archivos con el mismo nombre, reemplázalos.
4. Mantén tus imágenes/video si quieres usarlos, pero este diseño no depende de video para verse bien.
5. Verifica que GitHub Pages esté activo en Settings > Pages.

Nota:
Se eliminó el estilo verde neón/negro y se usó una línea visual clara con azul/violeta premium.
También se quitó el script publicitario externo que aparecía en el HTML anterior para dejar el sitio más limpio.

# Movve Wallet | Guía LATAM

Sitio web informativo y educativo enfocado en explicar **Movve Wallet** de forma clara, visual y paso a paso para usuarios de **República Dominicana, Colombia y LATAM**.

Este proyecto funciona como una guía de aprendizaje para conocer el ecosistema, revisar tutoriales, ver recursos de transparencia, acceder a PDFs informativos y entender los pasos principales antes de registrarse o avanzar dentro de la plataforma.

> Aviso: este sitio es una guía educativa. No constituye asesoría financiera. Toda inversión o participación en plataformas digitales implica riesgo.

---

## Objetivo del sitio

El objetivo principal es ayudar al visitante a entender:

- Qué es Movve Wallet.
- Cómo iniciar desde cero.
- Cómo crear y configurar su wallet.
- Cómo ver tutoriales y recursos educativos.
- Cómo revisar información de transparencia.
- Cómo acceder a PDFs, videos y enlaces importantes.

---

## Instalación local

Para ver el proyecto en tu computadora:

1. Descarga o clona el repositorio.
2. Abre `index.html` en tu navegador.
3. Para mejores resultados, usa un servidor local como Live Server en VS Code.

---

## Publicación en GitHub Pages

1. Sube todos los archivos a la raíz del repositorio.
2. En GitHub, entra a **Settings**.
3. Luego entra a **Pages**.
4. Selecciona la rama principal, normalmente `main`.
5. Guarda los cambios.
6. Espera unos minutos y abre tu dominio configurado.

## Optimización de rendimiento

Se recomienda mantener:

- Imágenes en formato WebP.
- Videos con `preload="none"`.
- Imágenes secundarias con `loading="lazy"`.
- Archivos CSS y JavaScript limpios.
- PDFs y videos pesados optimizados o alojados externamente si afectan la velocidad.
